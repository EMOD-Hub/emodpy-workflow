=== Editing Documentation ===

If you are going to be working on the docs, it is recommended that you do the following:

1) cd docs
2) pip install -r requirements.txt
3) cd ..
4) mkdocs serve
5) Open browser at http://127.0.0.1:8000/ and see documentation
6) Edit documentation and see it change in the browser.
